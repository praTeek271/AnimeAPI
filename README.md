# AnimeAPI
API for anime data
## Introduction

Welcome to the AnimeAPI repository! This repository contains an API for retrieving anime data. It allows you to perform CRUD operations on anime resources.

## Getting Started

To get started with the AnimeAPI, follow these steps:

1. Clone the repository: `git clone https://github.com/<your-username>/AnimeAPI.git`
2. Install the required dependencies: `pip install -r requirements.txt`
3. Run the server: `python manage.py runserver`


For detailed information on how to use each endpoint, refer to the [API documentation](/docs/api.md).
